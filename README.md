# JapVocRus
Проект для автоматической сборки учебных Anki-карточек из частотного списка японских слов.
Из csv файла по шаблону [слово,часть речи,чтение каной,частота] берет слова и получает переводы из Jardic. Есть возможность
генерировать чтения слов в формате wav, через ИИ, мы используем Kokoro-82M. Генерируются два набора карточек:

JP → RU (слово → перевод)

RU → JP (перевод → слово)

# Источники
NINJAL-LWP - https://tsukubawebcorpus.jp/en/ - список самых используемых японских слов.

Jardic - https://www.jardic.ru - Японо-русский онлайн-словарь Jardic.

Kokoro-82M - https://huggingface.co/hexgrad/Kokoro-82M - ИИ модель, выполняет для нас задачи TTS преобразований.
